üí° Testing Guide

  • Rich Context: Lots of descriptive text, should get accurate results
  • Medium Context: Some descriptive text, moderate accuracy expected
  • Sparse Context: Minimal text, tests API's ability with limited info
  • All Categories: Tests all gofakeit function categories at once
  • Category Selector: Test specific categories individually
  • Excluded Types: Checkboxes, radios, selects (no search API)
  • Custom Functions: Pre-assigned specific functions
